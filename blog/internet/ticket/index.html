    前端er是如何搶票的

    Kaye 3/18/2022 搶票

    香港第五波疫情來襲,深圳市於2022年1月26日修改政策,規定從香港返回入境中國大陸的旅客必須先在健康驛站上預約。由於入境人數眾多以及香港疫情蔓延至境內多地,深圳市一再收緊健康驛站入境名額,每日限額從原來的每日2000人下調至每日200。在港留學生紛紛表示一票難求,當中有不少人搶票多日無果,選擇向黃牛屈服,健康驛站的黃牛票一度被炒至4000港幣。我當時就身處香港見證著這種亂像,後來經過兩日人工搶票後,發現人手搶票基本上是不可能的。 只能說現代搶票完完全全是一埸網絡競賽,想要搶票,當然要靠科學 手段。

    # 搶票失敗的主要因素

    • web服務器時間與本機時間有差別
    • 人工作業手速慢

    # 服務器與本地時間較準

    搶票首先要把握好時機,在開放後第一時間完成操作。健康驛站對外宣布於每天早上10點開放搶票,而這個時間是依據服務器,而非客戶端的本地時間或是世界的標準時間。因此,搶票的第一步是要與服務器時間同步。我們可以通過請求查看服務器時間,再與自己的本地時間做比對。通過請求發現服務器的時區是GMT+0,而且換算成GMT+8時間後也與標準時間有大概有20秒的時間差。

    一般來說,搶票時間會分別在前端與後端設立校驗,通過前端擋掉一部分非法請求,防止服務器壓力過大。在健康驛站網頁裡定義了一個計時器,它控制客戶端是否可以跳轉到搶票的頁面。在搶票時間之前點擊搶票,本質上是不會發送請求的,我們的操作實際上會被前端擋掉。

    // 健康驛站源碼
// isOpen變量控制客戶端是否可以跳轉到搶票的頁面
    var timer = window.setInterval(function () {
        timeSpan = timeSpan - interval;
        anotherTimeSpan = anotherTimeSpan + interval;
        if (timeSpan <= 0 && timeSpan > reserveTimeSpan) {
            isOpen = true
        } else {
            isOpen = false;
        }
    }, interval);

    我們可以很輕鬆突破這層前端限制,只需要停用計時器,修改變量即可。在控制台輸入下面代碼,就可以突破前端的搶票時間限制。

    clearInterval(timer)
isOpen = true

    然而,即使突破前端限制,後端也會跟據服務器時間判斷,若果我們太早發請求,一樣會被拒絕。我們在上面已經發現服務器時間與標準時間有將近20秒的時間差,所以很有可能在10點前已經開放搶票了,而具體的開放時間需要不斷測試。

    # 人工作業手速慢,瀏覽器渲染需時

    人工作業手速慢這一點不需要我多說。使用瀏覽器搶票另一個問題是瀏覽器渲染需要一定時間,雖然所花的時間大體上都是毫秒級別,但是也有可能因此與票失之交臂。我們可以通過Node.js搭建一個簡易的搶票助手,它可以幫助我們精準控制請求時間,也可以免去瀏覽器渲染html,css的時間。

    # 分析

    搭建搶票助手可以分為三步:

    • 登錄
    • 查票
    • 訂票

    # 登錄

    機器登錄有很多種方法,最簡單的是通過Cookie繞過登陸。只要在請求的header帶上從瀏覽器複製下來的Cookie,就可以讓服務器以為我們已經登陸。(大家要保護好自己的cookie啊!!!)

    # 查票

    前後端分離項目一般會用接口進行溝通,只需要掉用相應的接口便可以查到余票。但是通過抓包并沒有發現接口,服務器只是返回一個HTML檔案以及相應的css﹑js檔案。请求余票的时候，返回的是HTML 代码，而不是我们预期的json数据,因此只能通過分析HTML提取我們想要的數據。

    # 訂票

    訂票是有相應的接口的,如果想要訂某一天的票,需要提供3個参數。訂票的接口是接受get請求,參數附用URL傳給服務器。

    "/passInfo/confirmOrder?checkinDate=" + checkinDate+"&t="+timespan+"&s="+sign

    搶票的步驟與方法都清晰了,接下來可以搭建我們的搶票助手了。

    class Ticket {
  constructor({ username, cookie, targetTime}) {
    this.cookie = cookie;
    this.username = username; // 用戶名
    this.targetTime = targetTime; //開搶時間
    this.times = 0; //已進行次數
    let stop = false;  // 是否停止搶票,防止外部直接更改變量
    this.getStop = function () {
      return stop;
    };
    this.setStop = function (ifStop) {
      stop = ifStop;
    };
  }

}

    # 定时任务

    我們的查票請求有可能會失敗,因此在開放搶票一瞬間需要不斷請求,直至成功訂票或是所以票被搶光為止。

    class SetInter {
    constructor({ timer, fn}) {
      this.timer = timer // 每隔幾秒执行一次
      this.fn = fn //callback
      this.rule = new schedule.RecurrenceRule();
      this.rule.second = this.setRule()
      this.init()
    }
    setRule() {
      let rule = [];
      let i = 1;
      while (i < 60) {
        rule.push(i)
        i += this.timer
      }
      return rule
    }
    init() {
      schedule.scheduleJob(this.rule, () => {
        this.fn()
      });
    }
  }
