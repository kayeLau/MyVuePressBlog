    用css 3D變換給奶荼手插上吸管

    Kaye 4/26/2022 3D transforms

    在 css 裡一切都是盒子,我們可以通過 over-flow 控制是否顯示溢出。如果您只希望顯示一側的溢出，該怎麼辦呢。

    試想像我們有一杯奶荼,要怎樣才能讓吸管穿過杯蓋,而不是簡單的疊在圖層上面或是下面。

    # 3D變換

    使用3D變換，您可以在 3D 空間中旋轉、變換和平移元素。要讓 3D 變換發揮作用，您需要兩個 CSS 屬性：

    • perspective: XXpx,指定了觀察者與 z=0 平面的距离,即鏡頭與物件的距离。
    • transform-style: preserve-3d, 告訴瀏覽器將元素保持在 3D 空間中。

    當視圖變為3D時後,我們可以操控元素的X,Y,Z三個維度,借此繪畫具有透視效果的圖像。

    回到插吸管的問題,3D變換後圖像具備了深度,我們可以讓杯蓋沿X軸旋轉一點,因為有perspective屬性，所以一切都以 3D 定位，因此，杯蓋頂部位於吸管之下，杯蓋底部位於在吸管之上。

    # Code

    <div class='css-container'>
    <div class='milk-tea threeD'>
        <div class='cap rotateX z-index-10'></div>
        <div class='strass'></div>
    </div>
</div>

    .css-container{
    padding:100px 10px;
    display:flex;
    align-items:center;
    justify-content: center;
    gap:80px;
}
.milk-tea{
    position:relative;
    width:100px;
    border-top: 200px solid #EDCFB3; 
    border-left: 20px solid transparent; 
    border-right: 20px solid transparent;
}
.milk-tea::before{
    position:absolute;
    z-index:20;
    content:'';
    background-image:url('./mxbc.png');
    background-size: contain;
    background-repeat: no-repeat;
    width:100px;
    height:80px;
    background-position:center;
    top: -140px;
}
.milk-tea::after{
    position:absolute;
    content:'';
    background-color:#EDCFB3;
    width:100px;
    height:50px;
    border-radius: 50px/25px;
    top:-25px;
}
.cap{
    position:absolute;
    top:calc(-200px - 25px);
    left:-23px;
    width:146px;
    height:50px;
    background-color:#C2EAEF;
    border-radius: 70px/25px;
}
.strass{
    position:absolute;
    width:10px;
    height:230px;
    background-color:#ccc;
    top:calc(-255px);
    left:50%;
    border-radius: 10px/5px;
}
.z-index-10{
    z-index:10;
}
.threeD{
  transform: perspective(3000px);
  transform-style: preserve-3d;
}
.rotateX{
    transform: rotateX(10deg);
}
